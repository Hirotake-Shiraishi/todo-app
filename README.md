# 完全版TODOアプリ（ドラッグ＆ドロップ・期限管理付き）

## 🔹 概要
ブラウザ上で動作する完全版のTODOアプリです。  
タスクの追加・完了・削除はもちろん、期限やカテゴリー管理、ドラッグ＆ドロップによる並び替え、期限切れタスクの赤色表示、localStorageでの永続化に対応しています。

## 🔹 機能
- タスクの追加 / 完了 / 削除
- 期限付きタスク管理（期限切れは赤色表示）
- カテゴリー管理（仕事 / プライベート / その他）
- 完了タスクの色分け
- 未完了→完了順、期限が近い順の自動ソート
- カテゴリーによるフィルタリング
- ドラッグ＆ドロップでタスク並び替え可能
- localStorageによるブラウザ内永続化

## 🔹 技術スタック
- HTML / CSS / JavaScript
- DOM操作・イベント処理
- 非同期処理（Promise / async/await）
- 配列操作（filter / sort / reduce / splice）
- localStorageで永続化

## 🔹 使い方
1. タスク名、期限、カテゴリーを入力
2. 「追加」ボタンまたはEnterキーでタスクを追加
3. タスクをクリックで完了/未完了切替
4. 削除ボタンで削除
5. ドラッグ＆ドロップでタスク順を変更
6. カテゴリーフィルタで表示を絞り込み可能
